  • discoverCharacteristics(_:)

    Function that triggers characteristics discovery for specified Services and identifiers. Discovery is called after subscribtion to Observable is made. - Parameter identifiers: Identifiers of characteristics that should be discovered. If nil - all of the characteristics will be discovered. If you’ll pass empty array - none of them will be discovered. - Returns: Observable that emits Next with array of Characteristic instances, once they’re discovered. Immediately after that .Complete is emitted.
